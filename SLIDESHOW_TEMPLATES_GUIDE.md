# Slideshow Templates - Complete User Guide

The slideshow template system allows you to save slideshow configurations and apply them to bulk image uploads, making it easy to create consistent slideshows with the same design and formatting.

## ðŸ“‹ How to Use Templates

### Creating a Template

1. **Build Your Slideshow**: Create a slideshow with all your desired settings
   - Add text overlays with titles, captions, and hashtags
   - Set the aspect ratio (9:16 for TikTok, 1:1 for Instagram, etc.)
   - Configure transitions and music settings

2. **Save as Template**: 
   - In the Slideshow Manager, click "Save as Template"
   - Enter a descriptive name (e.g., "3-Picture Story Template")
   - Add a description for future reference
   - Click "Create Template"

### Applying Templates to Bulk Images

#### Method 1: From the Image Uploader
1. **Upload Images**: Upload multiple images using the Image Uploader
2. **Select Images**: Click on images to select them (or "Select All")
3. **Apply Template**: Use the dropdown menu that appears:
   - "Apply to selected" - applies to currently selected images
   - "Apply to all" - applies to all uploaded images
4. **Choose Template**: Select from your saved templates
5. **Customize** (Optional): Modify title, caption, or hashtags
6. **Create**: The slideshow is automatically generated

#### Method 2: From Template Manager
1. **Open Template Manager**: Located in the Slideshow Manager section
2. **Click "Apply Template"**: Opens the template selection modal
3. **Choose Template**: Select from your saved templates
4. **Customize Settings**: Optionally modify title, caption, and hashtags
5. **Apply**: Creates the slideshow immediately

## ðŸ” Troubleshooting

### Common Issues

**Template Not Applying**
- Check that you have images selected
- Verify template exists and is not corrupted
- Ensure you have proper permissions

**Images Not Processing**
- Check image file formats (JPG, PNG supported)
- Verify internet connection for cloud processing
- Ensure images are not too large (>10MB each)

**Customizations Not Working**
- Clear browser cache and try again
- Check that required fields are not empty
- Refresh the page and reapply template

### Getting Help
- Check the browser console for error messages
- Verify all images are properly uploaded
- Ensure templates are saved correctly
